> **本檔性質**：PDF 列印與 CSV/Excel 匯出之**合規提醒**，**不構成法律意見**。

> **English version**: [`PRINT_EXPORT_LEGAL_NOTICE_EN.md`](./PRINT_EXPORT_LEGAL_NOTICE_EN.md)

## ⚠️ 重要：印出的單據與匯出的檔案具有法律與合規效力

erpilot v3.36 新增之列印 / 匯出功能涉及：
- **PDF 報價單 / 採購單 / 銷售單 / 出貨單**（對外文件）
- **客戶 / 供應商 / 料件清單**（含個資 / 商業機密）
- **銷售 / 採購訂單 / 庫存盤點 Excel**（財務數字依據）

這些檔案**錯誤或外洩可能引發**：
- 🔴 契約爭議（客戶依 PDF 主張價格 / 規格 / 交期）
- 🔴 個資外洩（《個人資料保護法》）
- 🔴 商業機密外洩（《營業秘密法》）
- 🔴 財報不實（內外部稽核依據）

**請務必依本檔指引使用，由經授權之人員簽核後對外。**

## 1. PDF 列印之法律邊界

### 1.1 報價單（print_quotation_pdf）

| 風險點 | 客戶責任 |
|---|---|
| 為要約之邀請 | PDF 上已自動加上「本報價單為要約之邀請，最終契約以雙方書面簽章為準」 |
| 簽章效力 | erpilot 之 PDF **不具電子簽章**；客戶需另以紙本印章 / 電子簽章服務（TWCA / SecureSign 等）完成簽核 |
| 金額正確性 | 客戶須於對外寄發前**人工覆核**金額、稅額、有效期 |
| 條款附件 | 商品規格、保固、付款條件等附件**不**由 erpilot 自動帶入；需自行附加 |

### 1.2 採購單 / 銷售單 / 出貨單

| 風險點 | 客戶責任 |
|---|---|
| 對外效力 | PDF 一經 email/列印交付供應商或客戶，即可能構成「要約」或「承諾」 |
| 修改不留痕 | erpilot 之 PDF **不記錄版本歷史**；修改後應重印並回收舊版 |
| 出貨點交 | 出貨單為「物權移轉」依據；簽收人須具代表權，否則可能無法主張交貨完成 |
| 稅務憑證 | 出貨單 / 銷售單**不等同**統一發票；發票須另依電子發票流程開立（見 v3.34 §1） |

### 1.3 共通免責

於適用法律所允許之最大範圍內，erpilot 對因下列情形所衍生之爭議**不承擔責任**：
- PDF 內容錯誤（金額 / 規格 / 交期）
- PDF 外洩、被竄改或被冒用
- 客戶因 PDF 而誤入契約義務
- 客戶未經授權對外寄發

## 2. CSV / Excel 匯出之資料保護

### 2.1 個資保護（PDPA）

匯出之**客戶清單**、**供應商清單**可能包含：
- 姓名、電話、Email（個資法第 2 條第 1 款「個人資料」）
- 公司地址、商業條件（營業秘密）

客戶於匯出後應：
- ✅ **加密儲存**（不要散落在桌面 / 共享資料夾）
- ✅ **限授權人存取**
- ✅ 對外寄送時**加密 Email** 或 **加密附件**
- ❌ **不**得轉售、轉讓、出借予第三方
- ❌ **不**得用於非業務目的（如行銷簡訊、其他公司之招攬）

### 2.2 商業機密保護

匯出之**料件清單**、**庫存盤點**、**銷售 / 採購訂單**含：
- 單價、成本、毛利（《營業秘密法》第 2 條）
- 客戶 / 供應商往來金額、付款條件

不當外洩可能構成：
- **民事**：損害賠償（《營業秘密法》第 12-14 條）
- **刑事**：5 年以下徒刑（《營業秘密法》第 13 條之 1）

### 2.3 對外送交（會計師、稅務代理）

| 場景 | 建議做法 |
|---|---|
| 寄給會計師事務所 | 加密壓縮 + 密碼分管道告知；不要 Email 明文 |
| 寄給稅務代理人 | 確認對方使用「會計師事務所專用 Email」 |
| 寄給銀行（聯徵 / 融資） | 用銀行之**安全上傳通道**；不要 Email 附件 |
| 內部跨部門 | 透過 erpilot 之**權限控制**直接查詢；避免反覆下載 |

## 3. 設定 / 示範資料（seed_demo_data_with_confirm）

### 3.1 不應在正式環境執行

| 場景 | 是否可用 |
|---|---|
| 安裝後 Day 1 試玩 | ✅ 可用（DB 空時） |
| 客戶 demo 環境 | ✅ 可用（隔離 DB） |
| **正式生產 DB** | ❌ **不應使用**（避免與正式資料混雜） |
| 教育訓練環境 | ✅ 可用（獨立 DB） |

erpilot 之 `seed_demo_data_with_confirm` tool 已內建保護：**DB 已有資料時自動 skip**。

### 3.2 載入後責任

- 示範資料僅供功能驗證，**不**反映實際商業關係
- 示範客戶 / 供應商 / 料件之**統編、地址、金額皆為虛構**
- 客戶不應將示範資料用於對外文件 / 對帳 / 申報

## 4. 範本與第三方授權

### 4.1 PDF 範本

- erpilot 內建 PDF 模板版型由 erpilot 自有 reportlab 程式碼產生
- **不**包含特定客戶之既有範本（如鼎新、正航之 PDF 樣式）
- 客戶如需匹配既有 ERP 版型，**自行客製化 print_service.py** 或由本團隊報價客製

### 4.2 字型授權

erpilot PDF 採系統字型 fallback 鏈：
1. Windows：微軟正黑體（**屬作業系統授權**）
2. macOS：PingFang（**屬 Apple 授權**）
3. Linux：Noto Sans CJK（**Open Font License**，免費可商用）
4. Fallback：Helvetica（**reportlab 內建 PostScript 字型**）

客戶於商業環境使用上述字型須遵守對應廠商之授權條款；erpilot 之程式碼**不**內嵌 / 重新散布上述字型檔案。

## 5. 紀錄保存與稽核

### 5.1 列印 / 匯出之 audit log

- erpilot 之 ConfirmCard 與 LLM tool 呼叫均**有 audit log**
- 列印 PDF（READ tier）**不**強制 audit；客戶如需紀錄誰印了什麼，需於 API gateway 層自行加 log
- 匯出 CSV/Excel **不**自動記錄「下載者」；客戶如需追蹤，建議於企業 SSO 層加紀錄

### 5.2 法定保存期

| 文件類別 | 法定保存期 | 法源 |
|---|---|---|
| 報價單 / 採購單 / 銷售單 | 至少 5 年 | 商業會計法 §38 |
| 統一發票（PDF 副本） | **至少 5 年** | 加值型及非加值型營業稅法 §32 |
| 庫存盤點報表 | 至少 5 年 | 商業會計法 §38 |
| 客戶 / 供應商主檔 | 視業務契約 | 各別契約條款 |

erpilot **不**自動歸檔 PDF；客戶須自行儲存（建議：每月匯出 + 雲端備份 + 加密）。

## 6. 免責條款（累積適用 v3.25.10 → v3.36）

於適用法律所允許之最大範圍內：

**1. 列印**
erpilot 對 PDF 內容之**法律效力、會計正確性、稅務合規性**不承擔責任；客戶應自行覆核並由有權人員簽核。

**2. 匯出**
erpilot 對匯出後檔案之**保管、傳輸、第三方處理**不承擔責任；客戶應依個資法、營業秘密法、契約義務自行控管。

**3. 範本**
erpilot 提供之 PDF 樣式僅為**功能示範**；客戶於正式對外文件需自行確認版型、條款、印章是否符合行業慣例與法律要求。

**4. 示範資料**
seed_demo_data_with_confirm 之內容**全為虛構**，不得用於對外契約 / 申報 / 對帳。
